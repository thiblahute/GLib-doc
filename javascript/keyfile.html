GLib.KeyFile.prototype.load_from_file 

function GLib.KeyFile.prototype.load_from_file(file: Number, flags: GLib.KeyFileFlags): {
    // javascript wrapper for 'g_key_file_load_from_file'
}

Loads a key file into an empty GLib.KeyFile structure.

If the OS returns an error when opening or reading the file, a G_FILE_ERROR (not introspectable) is returned. If there is a problem parsing the file, a G_KEY_FILE_ERROR (not introspectable) is returned.

This function will never return a GLib.KeyFileError.prototype.not_found error. If the file is not found, GLib.FileError.prototype.noent is returned.
