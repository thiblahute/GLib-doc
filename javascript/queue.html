GLib.Queue.prototype.free 

function GLib.Queue.prototype.free(): {
    // javascript wrapper for 'g_queue_free'
}

Frees the memory allocated for the GLib.Queue. Only call this function if queue was created with g_queue_new (not introspectable). If queue elements contain dynamically-allocated memory, they should be freed first.

If queue elements contain dynamically-allocated memory, you should either use GLib.Queue.prototype.free_full or free them manually first.
