GLib.Timer.prototype.start

    // javascript wrapper for 'g_timer_start'

Marks a start time, so that future calls to GLib.Timer.prototype.elapsed will report the time since GLib.Timer.prototype.start was called. g_timer_new (not introspectable) automatically marks the start time, so no need to call GLib.Timer.prototype.start immediately after creating the timer.

GLib.Timer.prototype.elapsed 

function GLib.Timer.prototype.elapsed(microseconds: Number): {
    // javascript wrapper for 'g_timer_elapsed'
}

If timer has been started but not stopped, obtains the time since the timer was started. If timer has been stopped, obtains the elapsed time between the time it was started and the time it was stopped. The return value is the number of seconds elapsed, including any fractional part. The microseconds out parameter is essentially useless.

Parameters

timer:

a GLib.Timer.

 GLib.Timer

return location for the fractional part of seconds elapsed, in microseconds (that is, the total number of microseconds elapsed, modulo 1000000), or null
