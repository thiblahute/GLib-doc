GLib.Hmac.unref 

def GLib.Hmac.unref (self):
    #python wrapper for 'g_hmac_unref'

Atomically decrements the reference count of hmac by one.

If the reference count drops to 0, all keys and values will be destroyed, and all memory allocated by the hash table is released. This function is MT-safe and may be called from any thread. Frees the memory allocated for hmac.

Structures

GLib.Hmac

An opaque structure representing a HMAC operation. To create a new GHmac, use g_hmac_new (not introspectable). To free a GHmac, use GLib.Hmac.unref.
