GLib.Timer.start 

def GLib.Timer.start (self):
    #python wrapper for 'g_timer_start'

Marks a start time, so that future calls to GLib.Timer.elapsed will report the time since GLib.Timer.start was called. g_timer_new (not introspectable) automatically marks the start time, so no need to call GLib.Timer.start immediately after creating the timer.

GLib.Timer.elapsed 

def GLib.Timer.elapsed (self, microseconds):
    #python wrapper for 'g_timer_elapsed'

If timer has been started but not stopped, obtains the time since the timer was started. If timer has been stopped, obtains the elapsed time between the time it was started and the time it was stopped. The return value is the number of seconds elapsed, including any fractional part. The microseconds out parameter is essentially useless.
